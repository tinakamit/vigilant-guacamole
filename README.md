# Nmap Network Scan Lab

## Project Overview
This project demonstrates a basic network scan using Nmap, a popular cybersecurity tool used for network discovery and security auditing.

## Objective
The goal of this lab was to practice using Nmap to identify open ports and understand how devices on a network can be analyzed.

## Tool Used
- Nmap

nmap -A scanme.nmap.org
