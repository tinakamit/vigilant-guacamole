# Nmap Network Scan Lab

## Project Overview
This project demonstrates a basic network scan using Nmap, a popular cybersecurity tool used for network discovery and security auditing.

## Objective
The goal of this lab was to practice using Nmap to identify open ports and understand how devices on a network can be analyzed.

## Tool Used
- Nmap

## Sample Command
```bash
nmap -A scanme.nmap.org
## What the Command Does
- Scans the target host
- Detects open ports
- Attempts service and version detection
- Provides detailed information about the target system

## Skills Demonstrated
- Network scanning
- Port identification
- Service enumeration
- Cybersecurity documentation

This project was completed for educational purposes using authorized testing targets.
